# Security Policy

## Supported versions

OffConvert is currently pre-1.0. Security fixes are applied to the latest
published release only.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use
[GitHub private vulnerability reporting](https://github.com/imustacho/offconvert/security/advisories/new)
and include the affected version, impact, reproduction steps, and any suggested
mitigation.

Avoid attaching private source media. A minimal synthetic sample is preferred
when a file is needed to reproduce the issue.
